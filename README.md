# Second Brain AI

多人共享 AI 第二大腦 MVP

## Stack

- Next.js 15
- TailwindCSS
- Supabase
- OpenAI / Claude / Gemini
- Vercel

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

複製 `.env.example`

```bash
cp .env.example .env.local
```

填入：

- Supabase URL
- Supabase Anon Key
- OpenAI API Key
- Anthropic API Key
- Gemini API Key

## Deploy

推薦直接部署到：

- Vercel
- Supabase
